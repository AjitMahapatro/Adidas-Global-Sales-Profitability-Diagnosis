# Adidas-Global-Sales-Profitability-Diagnosis
High-fidelity Power BI suite analyzing $899.9M in US retail revenue. Features AI root cause analysis, spatial bubble maps, and advanced DAX (Margin/Velocity). Engineered with a "Glass UI" to identify regional profit leakage and optimize product mix.  Tech Stack: Power BI, DAX, Data Modeling, Retail Strategy.
